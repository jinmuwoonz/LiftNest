# LiftNest 🏋️‍♂️

LiftNest is a modern, beautifully designed Flutter application to help weightlifters and fitness enthusiasts manage their gym inventories and quickly calculate the exact plate combinations needed for their target weights.

## ✨ Features

- **Inventory Management**: Create different inventories (e.g., "Home Gym", "Commercial Gym") and manage available plates and weights for each.
- **Workout Planning**: Organise workouts by day of the week, allowing you to quickly access your routine.
- **Smart Plate Calculator**: Enter your target weight, select your inventories (even pool them together!), and let the app calculate exactly which plates to load onto the bar.
- **Visual Barbell**: Instead of just text, see a colorful, visual representation of the barbell and the plates you need to load.
- **Dual Bar Support**: Calculate plates for a single heavy barbell or split the weight across two separate bars (like dumbbells).
- **Quick Calculator**: Instantly calculate plate combinations without having to create or save an exercise record.
- **Dark Mode Aesthetic**: A sleek, premium dark theme with vibrant orange accents designed specifically for gym environments.

## 🚀 Getting Started

To run this project locally, make sure you have Flutter installed.

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/liftnest.git
   ```
2. Navigate into the project directory:
   ```bash
   cd liftnest
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app (in debug mode):
   ```bash
   flutter run
   ```
5. Build the production-ready APK:
   ```bash
   flutter build apk
   ```
